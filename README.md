## DemoApp Instagram
In this demo app I created a Instagram application on Android Studio with Java. 
### App Contains
1. Page transition
2. Listing
3. Image Viewing

### In this demo app I used and practiced
- Firebase
- TextView
- ImageView
- ViewBingding
- ListView
- Mapping
- Adapter
- Model
